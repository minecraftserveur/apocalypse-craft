ceci est la page web du serveur apocalypse craft

sur apocalypse craft il y a de nombreuses commande que vous avez besoin de connaitre , et c'est ici que nous allons les lister

les commandes sur apocalyps craft
commadedescriptioninfo +
/warp "destination" vous permet de vous teleporter a des endroit predefini par le serveur il existe plusieur variant du /warp pour chaque endroit ou vous voulez aller : -/warp lobby pour aller au spawn -/warp arene pour aller en arene -/warp minage pour aller en minage -/warp aide pour avoir plus d'infos en jeux
/warps et /listwarp vous donne le nom de tous les warps en jeux dans le chat
/sethome "nom du home" place un home
/home "nom du home" vous permet de vous teleporter a un de vos home(s)
/listhome vous donne la liste de tous vos homes dans le chat
/delhome "nom du home" efface le home choisi
/tpa "nom du joueur auquel vous voulez vous tp" vous permet de vous teleporter a un joueur , le joueur qui recoit la demande doit taper la commande /tpaccept
/tpahere vous permet de teleporter un joueur a votre position , le joueur qui recoit la demande doit taper la commande /tpaccept
/tpdeny refuse une demande de tp
/back vous teleporte a l'endroit ou vous étes précédemment mort
